# Практическое задание по теме «Parallel LINQ» 

## Описание задания

Ниже представлен код программы, который загружает словарь английских слов.

```csharp
string wordLookupFile = Path.Combine(Path.GetTempPath(), "WordLookup.txt");

if (!File.Exists(wordLookupFile)) // Contains about 150000 words
  new WebClient().DownloadFile(
    "http://www.albahari.com/ispell/allwords.txt", wordLookupFile);
```

Расширьте этот код, добавив следующую функциональность:

- Поиск по шаблону: необходимо реализовать возможность поиска слов в словаре, которые соответствуют заданному регулярному выражению.
- Шифрование найденных слов: все слова, соответствующие заданному регулярному выражению, должны быть зашифрованы с использованием алгоритма AES.
- Использование Parallel LINQ: для повышения эффективности обработки больших объемов данных примените технологию Parallel LINQ. Это позволит распараллеливать процесс поиска и шифрования, обеспечивая более быструю обработку данных.
